🇫🇷 France phone-e ki korte hobe

Protita phone-e EK-BAR — tarpor ei dashboard-e nije chole ashbe.
  1. Developer Options + USB Debugging ON (Settings → About → Build number 7 bar tap → Developer options)।
  2. Tailscale app install (Play Store) → same account e login (upore account chip-e ja ache)।
  3. VPN permission chaile Allow। Tailscale "Connected" hobe।
  4. Wireless debugging on: phone USB cable diye PC-te lagao → Enable-Wireless-USB.bat double-click → tcpip 5555 on → cable khule felo।
  5. ⚡ Golden rule: phone Wi-Fi-te rakho (mobile data na)। Wi-Fi = direct = fast (~150ms France↔BD)। Mobile data = relay = slow। Card-e "direct"(sobuj)/"relay"(kmla) badge dekhbe।
  6. Byas — phone online hole card nije ashbe, "Control" e scrcpy khulbe।
Reboot dile tcpip 5555 chole jete pare → abar step 4। Rooted hole permanent।
